The aim is to build a web app that reads in data, draws graphs and charts with the data, and maybe a map too.

This project does things in small chunks, starting with a look a web servers...

1. Get data

Introduction to async programming

This is just the way things are on the Internet.
Async programming means your code doesn't stop dead while it waits for a slow resource to respond.

Explore various ways of getting data, including:

  • native JavaScript: fetch and XMLHttpRequest
  • jQuery: $.ajax({})
  • d3: d3.csv(input[, init][, row])
  • axios: axios.get(url[, config])

Also: importing and parsing csv files: PapaParse

2. Wrangle data

The joys of JSON

JSON is native to JavaScript, so is easy to deal with.
You just need to think serially

Do you even need to convert it into a data frame?

With JSON you iterate through data sequentially.
This is a different way of wrangling data, but you'll soon get the hang of it.

Here are some things that will help:

  • native JavaScript: Array iteration
  • lodash.js
  • d3.csvParse(string[, row])
